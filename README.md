# 👻 Flappy Bird X Kiro

A modern recreation of the classic **Flappy Bird** game built with **HTML, CSS, and JavaScript** using **Kiro IDE**. This project demonstrates game development fundamentals such as collision detection, physics, scoring, animations, and responsive user interaction.

## 🎮 Features

* Smooth Flappy Kiro gameplay
* Gravity-based bird movement
* Randomly generated obstacle pipes
* Collision detection
* Real-time score tracking
* Game Over screen
* Restart functionality
* Modern and responsive UI
* Sound effects (if enabled)

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Node.js (project tooling)
* Kiro IDE

## 🚀 Getting Started

### Prerequisites

* Node.js (recommended)
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/yashiro-nyx/Flappy-Bird-X-Kiro.git
```

Navigate into the project:

```bash
cd Flappy-Bird-X-Kiro
```

Install dependencies:

```bash
npm install
```

Run the project using your preferred development server.

If your project uses Vite:

```bash
npm run dev
```

Otherwise, simply open `index.html` in your browser or use a local web server.

## 🎯 How to Play

* Press **Space** or **Click** to make the bird flap.
* Avoid hitting the pipes.
* Don't touch the ground.
* Pass through pipes to earn points.
* Try to beat your highest score!
